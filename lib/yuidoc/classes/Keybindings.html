Keybindings Class

Defined in: keybindings.js:9

This is the main class Keybindings. It has important properties like the document body and all the active bindings, together with awesome methods.

The keys in the _bindings object is seperated with a / because you need two keys to get to that key, so you'll never get to bind a key with it (;, :, #... etc would also work).
